Languages

Python 3.11
Primary language for all DE work — producers, Spark jobs, dbt macros, Airflow DAGs. I use uv for dependency management and won't go back.
SQL (Snowflake)
Window functions, CTEs, QUALIFY for dedup. Snowflake's VARIANT type for semi-structured data is underrated. I write SQL before I write any Python.
PySpark 3.4
Structured Streaming for real-time, Spark SQL for batch. I write transformation logic as pure functions — makes unit testing actually possible.
Scala
For performance-sensitive Spark jobs where Python overhead matters. Also used for Kafka Streams applications.

Data Infrastructure

Apache Kafka
Confluent Cloud in production, Docker locally. Schema Registry with Avro is non-negotiable on any real project — JSON schemas are how you get paged at 3am.
Snowflake
My go-to warehouse. Auto-suspend set to 60s, CLUSTER BY for query optimization, Spark connector for streaming loads. SnowPro certified.
Apache Iceberg
For data lakehouse workloads on S3. Time-travel and schema evolution without breaking downstream consumers.
dbt
Every transformation goes through dbt. refs, tests, and docs are non-optional. Snowflake adapter with dbt-utils for generic tests.
Apache Airflow
Astronomer for managed deployments. TaskFlow API for cleaner DAG code. KubernetesPodOperator for isolation — no shared workers.
